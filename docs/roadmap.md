# Roadmap

Practical next steps for the project:

- deeper AST-based Solidity analysis instead of regex-only heuristics
- config and deployment sanity checks for trusted senders and routers
- richer report formats such as markdown and SARIF
- CI integration patterns for pull requests and release gates
- optional AI-assisted explanations for findings and remediation ideas
- more fixtures for edge cases and false-positive regression checks

The goal is to improve signal quality without turning the tool into a heavy compiler pipeline.
